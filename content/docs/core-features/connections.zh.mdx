# Connections

Connections 为 Dory 的其他能力定义了数据库上下文。不同环境通常需要不同的凭据、网络路径和访问策略。

## 连接检查清单

- 使用正确的数据库类型。
- 核对主机和端口。
- 确认当前是只读还是可写权限。
- 清晰标记环境名称。
