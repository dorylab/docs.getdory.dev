# Saved Queries

Saved Queries turn one-off analysis into reusable work. They are useful when a query needs to be rerun, shared with teammates, or used as the starting point for another investigation.

## When to save a query

- The query answers a recurring business question.
- The query took time to refine.
- A teammate will need the same analysis.
- The query feeds a chart or dashboard workflow.

## Good naming practice

Use names that describe the business outcome, not just the table name.
