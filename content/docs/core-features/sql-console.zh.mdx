# SQL Console

> Dory 的核心工作区。  
> 在一个界面内完成 **SQL 编写、AI 协作、结果分析与查询沉淀**。

![SQL Console](/images/sql-console.png)

## 概览

SQL Console 是大多数数据分析任务开始的地方，也是 Dory 最具代表性的工作区。

不同于传统数据库工具只提供 SQL 编辑器，Dory 将以下能力整合到同一工作流中：

- SQL 编写与执行
- Schema 浏览与表探索
- AI 生成 SQL 与优化 SQL
- 查询结果表格分析
- 图表快速可视化
- Saved Queries 管理
- 多标签页并行分析

这意味着你无需频繁切换多个工具，即可完成从问题到答案的全过程。

---

# 为什么选择 Dory SQL Console

## AI Native，而不是外挂 AI

许多工具只是提供一个聊天框，再让你复制 SQL 给 AI。

Dory 的 AI 深度集成在 SQL 工作流中，能够理解：

- 当前数据库连接
- 当前 Schema / Tables
- 当前标签页中的 SQL 内容
- 当前查询结果
- 当前分析上下文

因此生成结果更准确、更贴近真实数据场景。

你可以直接：

- 生成 SQL
- 修复报错 SQL
- 优化查询性能
- 解释复杂 SQL
- 根据结果继续分析
- 生成下一步查询语句

传统流程通常是：

SQL Client → 导出 CSV → 图表工具 → AI 工具 → 再回 SQL Client

Ask AI → 编辑 SQL → Run → 查看结果 → 图表分析 → 深入追问

全部在一个界面完成。

支持：

- 展开查看表
- 快速定位目标表
- 在分析前理解数据结构

对于大型数据库环境尤其高效。

# 2. 顶部操作栏

编辑区顶部提供高频操作入口：

<img
  alt="Top Panel"
  src="/images/top-panel.png"
/>

## Run

执行当前 SQL。

## Limit

限制返回行数，适合快速预览数据。

例如：

- 100
- 200
- 1000

避免一次返回过多数据。

## Ask AI

直接基于当前 SQL 发起 AI 协作。

## Save Query

保存当前查询，方便未来复用。

# 5. AI Copilot 面板

![ai-actions](/images/ai-actions.png)

右侧面板是 Dory 的核心能力之一。

支持三种工作模式：

## Ask

自然语言提问：

* 这段 SQL 在做什么？
* 为什么结果为空？
* 帮我按城市统计用户数

![ask](/images/ask.png)

## Action

执行明确任务：

* Rewrite SQL
* Optimize Query
* Fix Error
* Convert to JOIN

## Context

让 AI 理解当前工作环境：

* 当前数据库结构
* 当前表字段
* 当前标签页 SQL
* 当前查询结果

这让 AI 输出更可靠。
