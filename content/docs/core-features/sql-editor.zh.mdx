# SQL Editor

SQL Editor 是大多数分析任务的起点。Dory 会利用实时 Schema 信息增强自动补全、减少错误并提升迭代速度。

## 适合做什么

- 编写临时分析 SQL。
- 用自动补全验证表名和字段名。
- 在编辑与结果检查之间快速迭代。
- 结合当前查询上下文与 AI 协作。

## 常见流程

1. 选择当前连接。
2. 新建查询标签页。
3. 编写或粘贴 SQL。
4. 运行查询并查看结果。
5. 如果后续还会复用，就保存这条查询。
