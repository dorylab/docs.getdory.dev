# ClickHouse

ClickHouse is a core database target for Dory and typically benefits from the deepest feature coverage.

## Typical use cases

- Product analytics.
- Observability data.
- Event-heavy workloads.

## What to verify

- Driver compatibility.
- Network reachability.
- Permissions for metadata access.
