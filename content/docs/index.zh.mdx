# Dory 文档

Dory 是面向现代数据库的 AI 原生数据工作台。它将 Schema 感知的 SQL 编辑器、对话式 AI、数据库 Explorer、保存查询与图表分析整合在一个工作空间中。

## 文档内容

- 如何通过在线演示、桌面版、Homebrew 或 Docker 启动 Dory。
- 如何连接受支持的数据库，并使用 Explorer、SQL Editor 与图表能力。
- Dory 的 AI 功能如何基于数据库 Schema 与当前查询上下文工作。
- 当前版本在自托管、AI 配置与发布说明方面提供了哪些能力。

## 建议阅读顺序

1. 先看 [快速开始](./getting-started)。
2. 再配置 [数据库连接](./guides/connections)。
3. 了解 [SQL Editor](./guides/sql-editor) 与 [AI Chat](./guides/ai-chat)。
4. 团队部署前阅读 [自托管](./deploy/self-hosting)。

## 当前公开能力范围

根据公开仓库，Dory 当前重点覆盖：

- 对 ClickHouse 的深度集成与运维能力。
- PostgreSQL、MySQL、MariaDB、SQLite 支持。
- OpenAI、Anthropic、Google、Qwen、xAI 以及 OpenAI-compatible API 等 AI 提供商。

最近的产品变化请查看 [发布说明](./release-notes)。
