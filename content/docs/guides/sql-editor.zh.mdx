# SQL Console

SQL Console 是 Dory 最核心的工作区，用于编写、执行与迭代 SQL。

## 包含哪些能力

- 多标签 SQL 编辑。
- 基于真实 Schema 的自动补全。
- 多结果集支持。
- 生成、重写、解释 SQL 的 AI 操作。
- 基于结果集直接生成图表。

## 为什么重要

Dory 不只是一个查询执行器。它将 Schema 上下文、结果查看和 AI 辅助迭代放在同一处，减少在多个工具之间切换。

## 常见使用流程

1. 打开一个数据库连接。
2. 先用 AI 从自然语言生成 SQL，或直接手写 SQL。
3. 查看结果表格。
4. 如果会重复使用，就保存该查询。
5. 当结果更适合可视化时，直接生成图表。

## 当前预期

公开资料主要强调读分析场景与 SQL 编写效率。对于 AI 生成的语句，尤其是带副作用的写操作，仍应人工审查。
