# First Query in 3 Minutes

这是体验 Dory 价值最快的方式。完成本页步骤后，你将确认数据库连接正常、Schema 已加载，并完成第一轮查询分析。

## 开始之前

请先确保你已经完成：

- [Installation](./installation)
- [Connect Your Database](./connect-your-database)

## 1. 打开 Explorer 查看数据结构

进入 [Explorer](../core-features/explorer)，选择一个数据库与数据表。

建议优先查看：

- 用户表（users）
- 订单表（orders）
- 事件表（events）
- 日志表（logs）

这一步可以帮助你快速理解字段结构与可分析数据。

## 2. 打开 SQL Console

新建一个查询标签页，确认：

- 编辑器已连接当前数据库
- 表名可以自动补全
- 字段提示正常显示

如果这些能力正常，说明 Dory 已成功加载上下文。

## 3. 运行第一条安全查询

```sql
select *
from your_table
limit 20;
```

推荐先使用 `limit` 查询少量数据，快速验证：

- SQL 执行链路正常
- 表数据可访问
- 返回结果格式正确

## 4. 用 AI 生成下一步查询

打开 [AI Chat](../core-features/ai-chat)，例如输入：

- 帮我统计最近 7 天数据量趋势
- 找出增长最快的用户来源
- 按城市统计订单数量
- 生成适合柱状图展示的 SQL

此时 AI 会结合当前数据库结构生成更准确的 SQL。

## 5. 查看结果并转换图表

查询成功后，你可以切换到图表视图，快速生成：

- 折线图（趋势）
- 柱状图（分类对比）
- 饼图（占比结构）
- 表格（明细分析）

## 6. 保存你的第一条分析

如果这条查询后续还会复用，建议保存到 [Saved Queries](../core-features/saved-queries)。

这会帮助你逐步沉淀个人或团队的数据工作流。

## 常见问题

### 没有自动补全

请确认数据库连接成功，并等待 Schema 加载完成。

### 查询报错

请先检查表名、字段名和当前数据库是否正确。

### AI 结果不准确

连接真实数据库并加载 Schema 后，效果通常会明显提升。
