# First Query in 3 Minutes

这个流程可以最快确认你的连接、Schema 可见性和查询执行链路都已经正常工作。

## 1. 打开 Explorer

先在 [Explorer](../core-features/explorer) 中查看一个真实的数据表。

## 2. 打开 SQL Editor

新建一个查询标签页，观察自动补全是否能正确识别表和字段。

## 3. 运行一条安全查询

```sql
select *
from your_table
limit 20;
```

## 4. 让 AI 帮你改进

打开 [AI Chat](../core-features/ai-chat)，让它帮你补过滤条件、分组或生成适合图表展示的 SQL。

## 5. 保存结果

如果你后续还会复用这条分析，把它保存到 [Saved Queries](../core-features/saved-queries)。
