# Getting Started

You can evaluate Dory in three practical ways:

## 1. Use the live demo

Open `https://app.getdory.dev` and choose **Enter as Demo**. This is the fastest way to explore the interface without local setup.

## 2. Install a local desktop runtime

Use Homebrew on macOS:

```bash
brew install dorylab/dory/dory
```

Or download desktop builds from the GitHub releases page.

## 3. Run Dory with Docker

If you want a browser-based local deployment or you are evaluating self-hosting, use the Docker flow described in [Installation](./installation).

## What to do next
