# Quick Start

如果你希望用最少的准备快速体验 Dory，按这条路径开始即可。

## 方式 1：体验在线演示

打开 `https://app.getdory.dev`，选择 **Enter as Demo**。

## 方式 2：本地安装

在 macOS 上，可以直接通过 Homebrew 安装桌面版：

```bash
brew install dorylab/dory/dory
```

如果你更偏向容器部署或浏览器访问，继续阅读 [安装说明](./installation)。
