# First Query in 3 Minutes

This walkthrough is the fastest way to build confidence that your connection, schema visibility, and query execution path are all working.

## 1. Open Explorer

Start in [Explorer](../core-features/explorer) and inspect a real table.

## 2. Open SQL Editor

Create a new query tab and let autocomplete confirm that tables and columns are available.

## 3. Run a safe query

```sql
select *
from your_table
limit 20;
```

## 4. Ask AI to improve it

Open [AI Chat](../core-features/ai-chat) and ask for filters, grouping, or a chart-ready version.

## 5. Save the result

Store the query in [Saved Queries](../core-features/saved-queries) if you expect to reuse it.
