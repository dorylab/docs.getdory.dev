# Quick Start

Use this path if you want to see Dory working with minimal setup.

## Option 1: Try the live demo

Open `https://app.getdory.dev` and choose **Enter as Demo**.

## Option 2: Install locally

On macOS, install the desktop runtime with Homebrew:

```bash
brew install dorylab/dory/dory
```

If you prefer containers or browser access, continue to [Installation](./installation).

## What to do next

1. Complete [Connect Your Database](./connect-your-database).
2. Run your [First Query in 3 Minutes](./first-query-in-3-minutes).
3. Explore the [SQL Editor](../core-features/sql-editor) and [AI Chat](../core-features/ai-chat).
