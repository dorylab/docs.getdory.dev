# 介绍

Dory 是面向现代数据库的 AI 原生数据工作台。它把 Schema 感知的 SQL 编辑器、对话式 AI、数据库浏览、保存查询与图表分析整合在同一个工作空间中。

## 你可以用 Dory 做什么

- 连接 ClickHouse、PostgreSQL、MySQL、SQLite 等受支持数据库。
- 在写 SQL 之前先浏览表、字段和对象结构。
- 在编辑器中编写、运行并迭代 SQL。
- 使用带 Schema 上下文的 AI Chat 生成、解释和调试查询。
- 保存分析结果，并将查询结果转换为图表。
