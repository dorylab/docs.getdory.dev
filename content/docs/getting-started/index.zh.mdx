# 快速开始

你可以通过三种方式快速体验 Dory：

## 1. 使用在线演示

打开 `https://app.getdory.dev`，选择 **Enter as Demo**。这是无需本地安装即可体验界面的最快方式。

## 2. 安装本地桌面版

在 macOS 上可以直接使用 Homebrew：

```bash
brew install dorylab/dory/dory
```

也可以从 GitHub Releases 下载桌面安装包。

## 3. 使用 Docker 运行 Dory

如果你想在本地浏览器环境运行 Dory，或者正在评估自托管，请继续阅读 [安装说明](./installation)。
