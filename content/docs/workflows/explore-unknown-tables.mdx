# Explore Unknown Tables

When the schema is unfamiliar, start by inspecting names, types, and obvious relationships before asking AI for help.

## Suggested sequence

1. Open [Explorer](../core-features/explorer).
2. Inspect likely tables and columns.
3. Run a few limited preview queries.
4. Ask [AI Chat](../core-features/ai-chat) to explain relationships.
