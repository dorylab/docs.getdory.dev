# Generate SQL with AI

AI is strongest when you provide a concrete question, relevant tables, and the output shape you expect.

## Prompt template

- Goal: what question are you answering?
- Tables: which objects should be used?
- Filters: what time range or conditions matter?
- Output: what columns should be returned?
